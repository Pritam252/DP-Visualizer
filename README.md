# DP Visualizer

To build the project (Library + Demos):
run `cmake -G <Your build system generator> -B <build directory>`

Library target is `Visualizer`, and demo targets are `Visualizer1D` for 1D DP arrays and `Visualizer2D` for 2D DP arrays respectively.

**Note: the demos require input from console for the dimensions of the DP arrays. (1 for 1D, and 2 for 2D respectively)**
